 📁 File Organizer
 
A Python script that automatically organizes files in any folder into subfolders by type.
 
## Features
- Organizes Images, PDFs, Videos, and other files
- Works recursively inside subfolders
- Handles duplicate files (rename / skip / replace)
- Shows file count and size after organizing
- Removes empty folders automatically
## How to Use
1. Run the script
```
python file_organizer_final.py
```
2. Allow the program to access your files
3. Enter the folder path you want to organize
4. Choose which file types to organize
5. Confirm and done!
## Requirements
- Python 3.x
- No extra libraries needed
